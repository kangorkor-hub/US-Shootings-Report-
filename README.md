# US Shooting Dashboard Analysis

## Project Overview
The **US Shooting Dashboard Analysis** project explores patterns and trends in U.S. shooting incidents using Microsoft Excel. The objective of this project is to transform raw data into meaningful insights through data cleaning, analysis, and interactive dashboard design. The dashboard enables users to understand key trends related to demographics, geography, mental health indicators, and other factors associated with shooting incidents.

This project demonstrates skills in **data cleaning, exploratory data analysis (EDA), data visualization, and dashboard development** using Microsoft Excel.

---

## Project Objectives

The main objectives of this project were to:

- Clean and prepare the dataset for analysis.
- Identify trends in shooting incidents over time.
- Analyze shooting incidents across different demographic groups.
- Examine geographical patterns of shooting incidents.
- Investigate the relationship between mental health indicators and shooting incidents.
- Create an interactive dashboard that communicates insights effectively.

## Data Cleaning

The following data preparation tasks were performed:

- Removed duplicate records.
- Checked for missing values.
- Standardized text formatting using Power Query.
- Corrected inconsistent data entries.
- Created age groups for demographic analysis.
- Formatted date fields for time-based analysis.

---

## Dashboard Features

The dashboard provides an interactive overview of shooting incidents through:

- Total number of shooting incidents
- Annual shooting trends
- Shootings by race
- Shootings by age group
- Top affected states
- Distribution of fleeing status
- Distribution of mental health warning signs
- Interactive slicers for easier data exploration

---

## Key Insights

Some of the insights generated from the analysis include:

- Shooting incidents vary significantly across years.
- Certain states consistently record higher numbers of incidents.
- Most incidents involve specific age groups.
- The distribution of incidents differs across racial groups.
- Most individuals involved did not display recorded mental health warning signs.
- Fleeing status provides additional context for understanding incident patterns.
